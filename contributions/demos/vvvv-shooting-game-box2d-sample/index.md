1.  vvvv Shooting Game
This is a demo "30sec Shooting Game".
I think this also be a reference of Box2D.

This works on vvvv_45beta32.1(32bit) or later.

##  How to play
A,S,D,W Key : Move
Space Key : Shot a bullet
